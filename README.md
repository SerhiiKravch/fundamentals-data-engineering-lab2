# Лабораторна робота 2: Схеми "Зірка" та "Сніжинка"

Це проект з порівняння продуктивності схем "зірка" та "сніжинка" для аналізу даних про продажі.

## Структура проекту
- `data/` - генеровані тестові дані та бази даних SQLite
- `schemas/` - скрипти для створення схем бази даних
- `queries/` - аналітичні запити та порівняння продуктивності
- `requirements.txt` - залежності Python

## Концепції

### Схема "Зірка" (Star Schema)
- Денормалізована структура з центральною таблицею фактів
- Спрощені таблиці вимірювань (dimensions) з усіма атрибутами в одній таблиці
- Швидкі запити завдяки меншій кількості JOIN операцій
- Більше дублювання даних

### Схема "Сніжинка" (Snowflake Schema)  
- Нормалізована структура з центральною таблицею фактів
- Повністю нормалізовані таблиці вимірювань з підтаблицями
- Менше дублювання даних
- Більше JOIN операцій у запитах

## Запуск проекту

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Згенеруйте тестові дані:
```bash
python generate_data.py
```

3. Створіть схеми:
```bash
python schemas/star_schema.py
python schemas/snowflake_schema.py
```

4. Запустіть порівняльний аналіз:
```bash
python queries/performance_comparison.py
```
